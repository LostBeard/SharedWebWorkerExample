# SharedWebWorkerExample

Demo to provide aid for [SpawnDev.BlazorJS.WebWorkers issue #18](https://github.com/LostBeard/SpawnDev.BlazorJS.WebWorkers/issues/18)

## Overview
This is a simple Blazor WebAssembly application that demonstrates the use of Shared Web Workers using the SpawnDev.BlazorJS.WebWorkers library. 
The application consists of a main page that interacts with a Shared Web Worker to perform background tasks.
